# Task 3: Vulnerability Assessment Report (VAR)

## Objective

To perform reconnaissance, service enumeration, and vulnerability assessment on the OWASP Juice Shop web application using industry-standard cybersecurity tools.

## Tools Used

* Nmap
* Nikto
* Dirsearch
* Docker
* Kali Linux

## Work Performed

Performed vulnerability assessment on the OWASP Juice Shop application by conducting host discovery, service enumeration, and web security scanning. Used Nmap, Nikto, and Dirsearch to identify exposed services, security weaknesses, and accessible resources, and documented all findings with screenshots and scan reports.

## Key Findings

* Open service detected on Port 3000.
* Missing security headers identified.
* Accessible API documentation and resources discovered.
* Potential security misconfigurations observed.

## Skills Gained

* Network Reconnaissance
* Service Enumeration
* Vulnerability Scanning
* Directory Enumeration
* Security Reporting
